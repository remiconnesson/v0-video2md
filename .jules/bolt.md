## 2024-05-22 - [Virtual List Re-renders] **Learning:** `SlidesPanel` regenerates `feedbackMap` on every query update, causing all `SlideCard` items to receive new `initialFeedback` object references. This triggers O(N) re-renders in the virtual list when interacting with a single item. **Action:** Use `React.memo` on list items with a custom comparator that performs shallow equality checks on data props, and ensure callback props are stable.
